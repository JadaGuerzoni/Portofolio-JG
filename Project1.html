MailMate

A smart mailbox that combines sensors and a web application to keep users informed about their mail.

Year

Personal project one
Stack

Overview

MailMate is a smart mailbox concept that uses sensors to detect activity and a web dashboard to keep the user informed about incoming mail.

How it works

  • A Raspberry Pi handles the sensor input and system logic
  • The backend exposes mailbox state through a FastAPI service
  • The front end shows status updates and notifications in a simple web interface
  • MySQL stores mailbox events and system records for later review
